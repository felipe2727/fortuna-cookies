## Libraries Installed

I selected and installed the most robust libraries for modern, high-end web development, ensuring all are safe and production-ready:

1. **GSAP & `@gsap/react`**
   - **Purpose:** Industry-standard animation library. While Framer Motion is great for page transitions and simple UI interactions, GSAP excels at complex timelines, scroll-linked animations, and highly performant staggering. 
   - **Key Concept:** ScrollTrigger for dynamic scroll storytelling.

2. **Three.js, `@react-three/fiber`, & `@react-three/drei`**
   - **Purpose:** The absolute best tools for building 3D web experiences in React. 
   - **Key Concept:** We can add an interactive 3D fortune cookie, particles, or ambient floating backgrounds directly manipulated via React components.

3. **Lenis**
   - **Purpose:** Provides buttery smooth scrolling. Standard browser scroll feels clunky; Lenis hooks into the requestAnimationFrame loop to normalize scroll speeds and make the entire page feel like a premium, native application.
   - **Key Concept:** Smooth scroll hijacking that remains accessible and integrates seamlessly with GSAP's ScrollTrigger.

4. **Lucide React**
   - **Purpose:** Clean, modern, scalable SVG icons that are easily stylable with Tailwind CSS. Look professional without heavy monolithic icon libraries.

5. **Clsx & Tailwind-Merge**
   - **Purpose:** Together, these generate dynamic Tailwind classes without conflicts. Perfect for creating robust, reusable UI components (following the web artifacts builder and Radix UI patterns).
